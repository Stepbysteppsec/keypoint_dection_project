# point_regression

#### 介绍
pytorch实现热力图形式的关键点回归

#### 软件架构
pytorch+heatmap


#### 安装教程

1.  安装需要的库就可以，最新的都可以

#### 使用说明

1.  数据集图片存放地址：data/images     关键点xml存放地址：data/xml
2.  先运行utils生成data_center.txt文件
3.  开始训练 train.py
